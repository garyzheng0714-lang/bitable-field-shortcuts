飞书系统报图信息对接接口文档

接口概述

GetDesignList 接口用于获取展商图纸列表信息，包括展位基本信息和图纸审核状态。为了提高查询效率该接口只返回所有图纸都已提交的展商信息，由于展商图纸可能存在被驳回重新提交的情况，因此接口数据条数会有所变动，属于正常情况。

请求信息

请求URL: GetDesignList.ashx
请求方法: GET
注意：接口需要提供固定的code参数进行身份验证

请求参数

参数名 类型 是否必填 默认值 说明
code string 必填 - 身份验证码，固定值：70EA237E-0EDA-409C-A8C9-6BB22F01CE51
page int 选填 1 页码，最小值为1
limit int 选填 20 每页记录数，范围1-100，默认20
keyword string 选填 - 搜索关键词，用于模糊搜索展位号或展商名称

请求示例

GET http://service.best-expo.com.cn/GetUrl/lark/GetDesignList.ashx?code=70EA237E-0EDA-409C-A8C9-6BB22F01CE51&page=1&limit=10&keyword=A01

返回字段说明

根字段

字段名 类型 说明
code int 状态码，0表示成功，-1表示失败
message string 状态消息
count int 总记录数
data array 展商图纸列表数组

展商图纸对象字段

字段名 类型 说明
BoothID string 展位ID
BoothNO string 展位号
BoothExhibitor string 展商名称
BoothArea string 展位面积
Hall string 展馆
BoothTypeID string 展位类型ID
ShowNameCn string 展会名称
DesignUpStatus string 设计上传状态
DesignUpTime string 设计上传时间
DesignList array 图纸列表
DesignStats object 图纸统计信息

图纸对象字段

字段名 类型 说明
BDesignID string 图纸ID
BDesignType string 图纸类型ID
DesignCateCN string 图纸类型中文名称
UPStatus string 提交状态，1表示已提交
BDesignphoto string 图纸图片路径
DesignStatus string 综合审核状态

图纸统计对象字段

字段名 类型 说明
TotalCount int 已提交总图纸数量
UploadedCount int 已上传数量(UPStatus=1)
ApprovedCount int 已审核通过数量(BDesignStatus2=1)
PendingCount int 待审核数量(BDesignStatus2=0)
RejectedCount int 审核未通过数量(BDesignStatus2=2)

错误码说明

错误码 说明
0 成功
-1 参数错误或缺少参数
