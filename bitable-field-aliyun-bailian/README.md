# 阿里云百炼 (通义千问) 多维表格字段捷径
(Aliyun Bailian Field Shortcut for Bitable)

这是一个专为飞书/Lark 多维表格（Bitable）设计的字段捷径插件，深度集成了阿里云百炼（DashScope）API，让您可以在表格中直接调用强大的通义千问系列模型进行内容生成、数据分析和联网搜索。

## 📦 项目信息

- **项目名称**：阿里云百炼字段 (Aliyun Bailian Field)
- **GitHub 推荐仓库名**：`bitable-field-aliyun-bailian`

## ✨ 核心功能

1.  **多模型支持**：
    -   **Qwen-Flash**：快速、经济，适合简单任务。
    -   **Qwen-Plus**：均衡型，性价比首选。
    -   **Qwen3-Max**：超强能力，适合复杂推理和创作。
    -   **自定义模型**：支持输入其他阿里云百炼模型名称。

2.  **强大的联网搜索**：
    -   **标准搜索 (Turbo)**：快速响应（免费）。
    -   **深度搜索 (Max)**：全面信息检索（免费）。
    -   **Agent 搜索**：智能体模式，支持复杂任务和多源验证（0.004元/次）。

3.  **精准成本预估**：
    -   根据输入/输出 Token 数量及不同模型的费率自动计算预估费用。
    -   包含 Agent 搜索的额外调用费用。

4.  **详细用量统计**：
    -   直观展示总 Token、输入 Token 和输出 Token 的消耗情况。

## 🛠 安装与使用

### 1. 环境准备
确保您已安装 Node.js 环境。

### 2. 安装依赖
```bash
npm install
```

### 3. 本地调试
```bash
npm run start
```
启动后，在飞书/Lark 多维表格中打开「字段捷径调试助手」，即可加载并测试插件。

### 4. 打包发布
```bash
npm run pack
```
执行后会在 `output` 目录下生成 `output.zip` 文件，可上传至多维表格开发者后台进行发布。

## ⚙️ 字段配置说明

在多维表格中添加此字段后，配置项如下：

| 配置项 | 说明 |
| :--- | :--- |
| **提示词 (Prompt)** | 选择表格中的文本字段作为输入提示词。 |
| **选择模型 (Model)** | 选择要使用的模型（Flash/Plus/Max/Custom）。 |
| **其他模型名称** | 当选择“其他模型 (自定义)”时生效，填入具体的模型代码（如 `qwen-long`）。 |
| **联网搜索配置** | 选择搜索策略：<br>- **不开启**：仅使用模型知识。<br>- **标准/深度**：免费联网搜索。<br>- **Agent**：按次收费的高级搜索。 |
| **API Key** | 输入您的阿里云 DashScope API Key (`sk-...`)。<br>[如何获取 API Key?](https://bailian.console.aliyun.com?tab=model#/api-key) |

## 📊 返回数据结构

插件执行成功后，将写入以下结构化数据：

- **输出结果**：模型生成的文本结果。
- **思考过程**：模型输出的思维链内容（如 DeepSeek-R1 等推理模型）。
- **预估费用 (元)**：本次调用的预估成本（保留4位小数）。
- **Token 消耗**：格式为 `总数 (入:输入数 / 出:输出数)`。
