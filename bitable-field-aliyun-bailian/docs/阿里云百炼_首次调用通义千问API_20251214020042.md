## 三、获取 API Key

前往 API Key 管理页面创建 API Key。

**建议：**
- API Key 归属业务空间选择「主账号空间」
- 使用子空间 API Key 时，需由主账号为子空间授权模型访问权限

---

## 四、配置 API Key 到环境变量（推荐）

将 API Key 配置到环境变量，避免在代码中明文写 Key，降低泄漏风险。

---

### macOS / Linux（Bash）

#### 永久生效
```bash
echo "export DASHSCOPE_API_KEY='YOUR_DASHSCOPE_API_KEY'" >> ~/.bashrc
source ~/.bashrc
```

#### 临时生效
```bash
export DASHSCOPE_API_KEY="YOUR_DASHSCOPE_API_KEY"
```

#### 验证
```bash
echo $DASHSCOPE_API_KEY
```

---

### macOS（Zsh）

```bash
echo "export DASHSCOPE_API_KEY='YOUR_DASHSCOPE_API_KEY'" >> ~/.zshrc
source ~/.zshrc
echo $DASHSCOPE_API_KEY
```

---

### Windows（系统环境变量）

- 变量名：`DASHSCOPE_API_KEY`
- 变量值：你的 API Key

CMD 验证：
```cmd
echo %DASHSCOPE_API_KEY%
```

PowerShell 验证：
```powershell
echo $env:DASHSCOPE_API_KEY
```

## 六、Python 示例

### 1. 环境要求

- Python ≥ 3.8
- pip 可用

```bash
python -V
pip --version
```

---

### 2. 安装 SDK

#### OpenAI Python SDK（OpenAI 兼容方式，推荐）
```bash
pip install -U openai
```

#### DashScope Python SDK
```bash
pip install -U dashscope
```

---

### 3. Python（OpenAI 兼容方式）

```python
import os
from openai import OpenAI

client = OpenAI(
    api_key=os.getenv("DASHSCOPE_API_KEY"),
    base_url="https://dashscope.aliyuncs.com/compatible-mode/v1"
)

completion = client.chat.completions.create(
    model="qwen-plus",
    messages=[
        {"role": "system", "content": "You are a helpful assistant."},
        {"role": "user", "content": "你是谁？"}
    ]
)

print(completion.choices[0].message.content)
```

---

### 4. Python（DashScope SDK）

```python
import os
from dashscope import Generation

messages = [
    {"role": "system", "content": "You are a helpful assistant."},
    {"role": "user", "content": "你是谁？"}
]

response = Generation.call(
    api_key=os.getenv("DASHSCOPE_API_KEY"),
    model="qwen-plus",
    messages=messages,
    result_format="message"
)

print(response.output.choices[0].message.content)
```

### 3. Node.js（OpenAI 兼容方式）

```js
import OpenAI from "openai";

const openai = new OpenAI({
  apiKey: process.env.DASHSCOPE_API_KEY,
  baseURL: "https://dashscope.aliyuncs.com/compatible-mode/v1"
});

const completion = await openai.chat.completions.create({
  model: "qwen-plus",
  messages: [
    { role: "system", content: "You are a helpful assistant." },
    { role: "user", content: "你是谁？" }
  ]
});

console.log(completion.choices[0].message.content);
```

## 十、常见问题

### 1. 免费额度用完后如何处理？
确保账号无欠费，模型调用将自动扣费，按小时出账。

### 2. 报错 `Model.AccessDenied`
通常原因：
- 使用了子业务空间 API Key
- 子空间未被主账号授权模型访问权限
