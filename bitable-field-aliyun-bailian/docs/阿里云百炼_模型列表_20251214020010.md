# 阿里云百炼 · 模型列表与计费说明（参考文档）

> 来源：阿里云帮助中心 · 大模型服务平台百炼（Model Studio）

> 不保证与官网实时同步，仅用于模型选型、能力对比、计费理解。

---

## 一、平台概述

阿里云百炼（Model Studio）提供通义系列大模型及第三方模型，覆盖：

- 文本生成（LLM）
- 多模态（图像 / 音频 / 视频）
- 语音识别与合成
- 图像 / 视频生成与编辑
- 向量化（Embedding）
- 行业 / 领域模型

支持 **API 调用、Batch 推理、上下文缓存、思考模式（Thinking）** 等能力。

---

## 二、旗舰模型（Qwen 商业版）

### 1. 通义千问 Max
- 定位：复杂、多步骤、高推理任务
- 最大上下文：262K
- 特点：能力最强、支持 Search Agent

### 2. 通义千问 Plus
- 定位：能力 / 成本 / 速度均衡
- 最大上下文：1M
- 支持：思考模式 / 非思考模式

### 3. 通义千问 Flash
- 定位：简单任务、极低成本、高速度
- 最大上下文：1M
- 适合：高并发、轻量调用

### 4. 通义千问 Coder
- 定位：代码与工具调用
- 特点：Agent 能力强，适合编程场景

---

## 三、文本生成模型（重点）

### Qwen3-Max
- 稳定版 / 快照版 / Preview（思考）
- 上下文：262,144
- 计费：**阶梯计价（随输入 Token 增加）**
- 支持：
  - Batch 半价
  - 上下文缓存折扣

### Qwen-Plus（Qwen3）
- 上下文：最高 1,000,000
- 模式：
  - 思考模式（Thinking）
  - 非思考模式
- 适合：中高复杂推理、Agent 系统

### Qwen-Flash（Qwen3）
- 上下文：1,000,000
- 极低输入 / 输出成本
- 支持 Batch / Cache

---

## 四、推理模型

### QwQ
- 基于 Qwen2.5 强化学习训练
- 特点：
  - 数学 / 代码推理能力极强
  - 对标 DeepSeek-R1 满血版
- 适合：
  - 数学题
  - 逻辑推理
  - 复杂算法分析

---

## 五、超长上下文模型

### Qwen-Long
- 最大上下文：**10,000,000 Tokens**
- 适合：
  - 超长文档分析
  - 合同 / 报告 / 法规
  - 批量摘要与信息抽取
- 成本：极低（支持 Batch 半价）

---

## 六、多模态模型

### Qwen-Omni
- 输入：文本 / 图片 / 音频 / 视频
- 输出：文本 / 音频
- 应用：
  - 视觉理解
  - 语音助手
  - 多模态 Agent

### Qwen3-Omni-Flash（推荐）
- 新一代 Omni 模型
- 支持：
  - 思考 / 非思考
  - 多语言
  - 多音色（最多 49 种）
- 替代：Qwen-Omni-Turbo（已不推荐）

---

## 七、实时多模态

### Qwen-Omni-Realtime
- 支持音频流式输入
- 内置 VAD（语音活动检测）
- 适合：
  - 实时语音对话
  - 实时翻译
  - 会议助手

---

## 八、计费与规则要点

- 计费单位：Token（输入 / 输出）
- 商业版模型：
  - 新用户一般有 **100 万 Token 免费额度（90 天）**
- 影响价格的因素：
  - 上下文长度
  - 是否思考模式
  - 是否 Batch 调用
  - 是否使用上下文缓存
