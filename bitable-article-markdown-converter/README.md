# 项目简介
- 将网页正文一键提取并转换为 Markdown 的工具，适配飞书字段捷径（Feishu/Lark Field Shortcut）。
- 输入任意网页链接（含微信公众号文章），输出干净、可阅读的 Markdown 文本。

# 核心特性
- **稳定输出**：优先使用 r.jina.ai 抓取解析；失败或效果差时，回退为直抓 HTML + 自研正文抽取。
- **正文抽取**：基于简化 DOM 树与内容评分算法，定位主内容区，避免导航、侧栏、评论等噪音。
- **Markdown 转换**：保留 h1–h6 标题、链接与图片，自动处理换行和空行。
- **标题处理**：从 og:title/h1/title 提取标题，并与正文首行去重、规范化。
- **微信图片修复**：微信图床链接转为 HTTPS，并通过 images.weserv.nl 代理绕过防盗链；GIF 可保留动效。
- **噪音过滤**：清理推广、社交按钮、二维码、装饰线等常见干扰项，输出更聚焦。
- **兼容性**：构建产物为 CommonJS，避免 VM 中 “import/export 仅在 ESM 可用” 的错误。

# 使用方法
- 开发调试：`npm run dev` 或 `npm run start`
- 构建产物：`npm run build`
- 打包发布：`npm run pack` （生成 zip 包到 output/，上传至目标平台）

# 适用场景
- 快速归档网页文章为 Markdown
- 飞书表单/字段中将 URL 转换为结构化内容
- 资讯采集、知识库沉淀、阅读优化

# 说明
- 不依赖 jsdom/readability/turndown 等第三方库，构建体积小、运行稳定。
- 对多数资讯/文章页表现良好；极度定制化页面效果取决于源页面结构和抓取通道。
