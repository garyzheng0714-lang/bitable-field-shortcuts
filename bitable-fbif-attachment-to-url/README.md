# 附件转链接 (Attachment to URL)

这是一个飞书多维表格字段捷径（Field Shortcut），用于将多维表格中的附件自动上传到对象存储（阿里云 OSS 或火山引擎 TOS），并生成永久可访问的 URL 链接。

## ✨ 核心功能

*   **自动上传**：将多维表格附件列中的文件自动上传到指定的对象存储桶。
*   **支持多云**：同时支持阿里云 OSS 和火山引擎 TOS。
*   **自定义命名**：
    *   支持使用原文件名或自定义文件名。
    *   支持变量替换（如 `{name}` 原文件名、`{date}` 日期）。
    *   支持自动添加时间戳后缀。
*   **防冲突机制**：采用 `fbif-attachment-to-url/{YYYY}/{MM}/{tableId}_{随机串}_{时间戳}/{filename}` 的路径结构，确保文件永不覆盖，且易于溯源。
*   **批量处理**：支持一次性处理多个附件。

## 🛠 使用指南

### 1. 配置参数

在多维表格中添加此字段捷径后，需要配置以下参数：

*   **附件字段**：选择包含要上传文件的附件列。
*   **文件命名**：(可选) 自定义文件名的规则。
*   **文件命名是否加上时间戳**：选择“是”或“否”。
*   **存储类型**：选择“阿里云 OSS”或“火山引擎 TOS”。
*   **AccessKeyId / Secret / Bucket / Region**：填写对应云厂商的访问凭证和存储桶信息（可留空使用默认配置）。

### 2. 输出结果

执行成功后，该字段将返回上传后的文件 URL。如果是多个文件，URL 将以逗号分隔。

## 📦 存储路径规范

为了保证数据的安全性和唯一性，文件上传路径遵循以下规范：

```text
fbif-attachment-to-url/2026/02/tbl123456_JmpFpoGDozrg5ZGT381BDg_1706857200000/example.jpg
```

*   **前缀**：`fbif-attachment-to-url`
*   **日期归档**：按年/月归档（如 `2026/02`）
*   **批次隔离**：`{tableId}_{16位高熵随机串}_{毫秒级时间戳}`，确保每次运行的绝对隔离。

## 🚀 开发与部署

### 本地运行
1.  安装依赖：`npm install`
2.  启动服务：`npm start`
3.  本地测试：`npm run dev`

### 打包发布
运行 `npm run pack` 生成 `output/output.zip`，然后在飞书开发者后台上传发布。
