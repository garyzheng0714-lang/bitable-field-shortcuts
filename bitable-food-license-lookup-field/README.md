# Workspace 项目说明

## 项目简介

该项目是基于飞书多维表格的“字段捷径”扩展，用于通过食品生产许可证编号实时查询许可证详情（如生产者名称、许可证编号、社会信用代码、监督管理机构、有效期等），便于在业务表格中快速回填与校验。

## 功能特性

- 输入食品生产许可证编号，自动完成格式规范化与校验
- 调用阿里云市场食品许可证查询接口，获取并映射关键字段
- 结果包含生产者名称、许可证编号、社会信用代码、监督管理信息、地址、有效期与状态等

## 安装指南

1. 克隆仓库
2. 安装依赖
3. 配置环境
4. 启动项目

详细步骤：
1) 克隆仓库：使用 Git 将本仓库克隆到本地
2) 安装依赖：执行 `npm install`
3) 配置环境：
   - 在 `config.json` 中填写阿里云市场订阅得到的 AppCode（建议不要提交真实密钥到仓库）
   - 也可将敏感信息配置为 `.env` 并确保 `.gitignore` 已忽略
4) 启动项目：
   - 开发：`npm start` 或 `npm run dev`
   - 构建：`npm run build`
   - 打包：`npm run pack`

## 使用说明

- 在飞书多维表格中安装并添加此字段捷径
- 选择包含食品许可证编号（文本类型）的字段
- 输入示例：`SC110230521044400`（无空格与特殊字符）
- 扩展会自动查询并将结果字段回填到对象结果中
