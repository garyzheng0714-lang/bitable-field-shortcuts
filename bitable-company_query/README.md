# 企业工商信息查询字段插件

## 项目简介

这是一个基于飞书多维表格的字段插件，通过企业名称自动查询并返回详细的工商信息。插件支持选择现有字段作为输入源，自动调用第三方API获取企业的统一社会信用代码、法定代表人、注册资本、经营范围等完整工商信息。

## 功能特性

- **智能字段选择**：支持选择现有的文本字段作为企业名称输入源
- **自动文本清理**：智能去除输入中的空白字符、占位符和特殊符号
- **完整工商信息**：返回包含25+字段的详细企业工商信息
- **友好错误提示**：提供中文错误信息，帮助用户快速定位问题
- **实时数据查询**：基于阿里云API市场的企业工商信息精准查询服务

## 返回信息字段

- 基本信息：公司名称、曾用名、英文名称
- 注册信息：统一社会信用代码、组织机构代码、注册号、税号
- 法人信息：法定代表人、法人类型、企业状态
- 时间信息：成立时间、核准日期、营业期限
- 资本信息：注册资金、注册地址
- 业务信息：经营范围、所属行业
- 联系信息：电话、邮箱、网址
- 地理信息：省、市、区

## 安装指南

1. **克隆仓库**
   ```bash
   git clone <repository-url>
   cd field-demo
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **配置环境**
   - 确保已安装 Node.js 14+ 版本
   - 配置飞书开发者账号和应用权限

4. **启动项目**
   ```bash
   # 开发模式
   npm start
   
   # 构建项目
   npm run build
   
   # 打包插件
   npm run pack
   ```

## 使用说明

1. **添加字段**：在多维表格中添加此插件字段
2. **选择数据源**：选择包含企业名称的现有字段
3. **自动查询**：插件将自动调用API查询企业工商信息
4. **查看结果**：在字段中查看返回的详细企业信息
