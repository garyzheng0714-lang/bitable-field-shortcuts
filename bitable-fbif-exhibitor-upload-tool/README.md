# 【FBIF】展商报馆信息上传工具 (FBIF Exhibitor Upload Tool)

这是一个基于飞书多维表格字段捷径（FaaS）开发的工具，用于将飞书多维表格中的展商数据一键推送至 FBIF 展商报馆系统。

## 功能特性

- **字段映射**：在插件配置界面直接将多维表格字段映射到接口参数，无需修改代码。
- **一键推送**：支持将单行展商数据实时推送至指定接口。
- **自动校验**：内置严格的字段必填校验和格式验证（如邮箱格式、展位号字符限制）。
- **结果反馈**：实时返回推送状态、接口响应码及完整响应信息，方便排查问题。
- **多语言支持**：支持中文、英文、日文界面。

## 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 启动本地服务

```bash
npm start
```

*注意：如果遇到 `@bdeefe/feishu-devtools-core` 模块缺失错误，请确保已在飞书客户端安装并运行“字段捷径调试助手”。*

### 3. 调试插件

1. 打开飞书多维表格，在侧边栏启用“字段捷径调试助手”。
2. 连接本地启动的服务（通常为 `http://localhost:8080`）。
3. 在表格中添加一个调试字段，选择本插件。
4. 在配置面板中填写或选择对应的字段映射：
   - **必填项**：展位号、展馆编号、公司名称、展位面积、登录账号、登录密码、联系人姓名、联系人邮箱。
   - **可选项**：公司名称（英文）、展位类型、语言。
5. 点击“调试字段”按钮进行测试。

## 发布

执行以下命令打包插件：

```bash
npm run pack
```

生成的 `output/output.zip` 文件即可用于上传至飞书开发者后台进行发布。

## 接口对接说明

本插件对接接口：`http://service.best-expo.com.cn/geturl/lark/UpExhibitors.ashx`

### 参数映射关系

| 插件配置项 | 接口参数 | 说明 |
| --- | --- | --- |
| 展位号 | `booth_no` | 仅允许英文、数字、下划线、连字符 |
| 展馆编号 | `hall` | |
| 公司名称（中文） | `company_name` | |
| 公司名称（英文） | `company_name_en` | |
| 展位面积 | `area` | 必须为数字 |
| 登录账号 | `username` | 唯一标识 |
| 登录密码 | `password` | |
| 展位类型 | `booth_type` | RS (光地) / SS (标摊) |
| 联系人姓名 | `contact_name` | |
| 联系人邮箱 | `contact_email` | 需符合邮箱格式 |
| 语言 | `lang` | CN (中文) / EN (英文) |
