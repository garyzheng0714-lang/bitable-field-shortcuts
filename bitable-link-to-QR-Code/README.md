# 链接转二维码 (Link to QR Code)

这是一个多维表格字段捷径插件，用于将文本或链接转换为二维码图片，并支持选择特定的二维码样式（如带Logo的样式）。

## 功能特点

- **支持多种预设样式**：内置了 D10-D25 等多种特定业务场景的样式（FBIF, Simba 等）。
- **自定义样式支持**：允许用户输入自定义的样式 ID。
- **自动代理**：使用 `wsrv.nl` 进行图片代理，解决跨域和 Content-Type 问题，确保图片能正确写入多维表格附件。
- **简单易用**：只需选择样式并输入内容，即可生成二维码。

## 开发与运行

1.  **安装依赖**
    ```bash
    npm install
    ```

2.  **启动本地服务**
    ```bash
    npm start
    ```

3.  **打包发布**
    ```bash
    npm run pack
    ```
    运行后会生成 `output/output.zip`，可用于上传至飞书开发者后台。

## 注意事项

- 本插件依赖公网环境访问 `open-api.cli.im` (草料二维码) 和 `wsrv.nl` (图片代理)。
- 确保运行环境支持访问外网。
